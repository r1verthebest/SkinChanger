# SkinChanger
SkinChanger é um plugin sênior para Spigot 1.8.8 focado em performance. Utiliza HikariCP para persistência MySQL assíncrona e NMS para atualização instantânea de skins via pacotes. Oferece menus de alta fluidez carregados via Base64 (Zero-Lag GUI) e uma biblioteca extensível via YAML. Otimizado para redes competitivas e escaláveis.
